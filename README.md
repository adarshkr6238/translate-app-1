**Translate App** is a fast, lightweight, and floating Android translator. It uses Optical Character Recognition (OCR) to instantly capture and translate text right on your screen without switching apps!

## ✨ Features
- **💬 Floating Bubble:** Draggable overlay for 1-tap translation anywhere.
- **👁️ Smart OCR:** Automatically detects text on your screen using Google ML Kit.
- **🔄 Multi-Provider Support:** Choose between Offline (ML Kit) or Online (Lingva, MyMemory, SimplyTranslate).
- **🌍 Language Override:** Force a specific source language for accurate translations when multiple languages are visible.
- **🌙 Dark Mode:** Clean Material 3 design that respects your system theme.
